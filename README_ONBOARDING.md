## ⚡ Sürətli Start

### İndi Test Et:
```bash
npm run dev
# Və ya
npm run build && npm run preview
```

Səhifəni aç → Onboarding göstərilir  
F5 bas (yenilə) → Yenə göstərilir ✅

### Production-a Çıx (Sonra):
1. `src/App.tsx` aç
2. `testMode={true}` → `testMode={false}` dəyişdir
3. `npm run build && npm run deploy`
